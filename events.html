For Centers

Submit or manage an event.

In the real app, a center manager will log in, add event details, and submit changes for review if needed.

1. Choose host center

Events connect to a center profile, so visitors can see who is hosting and whether the profile is claimed.

2. Add event details

Date, time, format, location, price, registration link, description, photo, facilitator, and contact.

3. Publish

Approved events appear on the global calendar, the host center profile, and relevant location/type searches.
